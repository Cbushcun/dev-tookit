# What this repo is

- A collection of code snippets. Categorized in various folders as displayed.
- More information on each groups of snippets can be found in the respective directory README files.
- Some of the snippets within the repo may rely on other local/relative snippets within it, therefore copying snippets to other projects may require a deeper extraction.
- Overall, the goal is to produce a reference library of snippets to use in future projects.

# How to use the repo

- Folders are named after the category of the snippets assigned to them.
- Each folder contains a requirements.txt file for package reproduction
